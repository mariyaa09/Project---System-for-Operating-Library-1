Описание на проекта:

Този проект представлява конзолно приложение на C#, разработено във Visual Studio, което реализира система за управление на библиотека. Целта е да се демонстрират основните обектно-ориентирани принципи (ООП) – наследяване, полиморфизъм, инкапсулация и обработка на изключения.
Проектът е подходящ за учебна цел и защита.

 Цел на проекта:
Управление на библиотека с книги
Създаване на потребителски профили
Заемане и връщане на книги със срок
Обработка на грешки чрез изключения
Демонстрация на ООП принципи

Структура на проекта:
LibraryManagementSystem
│
├── Program.cs
├── Book.cs
├── Library.cs
├── User.cs
├── Student.cs
└── Librarian.cs

Потребители
 User (абстрактен клас)
Име на потребителя:
Абстрактни методи:
BorrowBook()
ReturnBook()
Student
Може да заема и връща книги
Librarian
Може да добавя книги
Няма право да заема или връща книги

Книги:
Всяка книга съдържа:
Заглавие
Автор
Статус (налична / заета)
Срок за връщане (14 дни)
При заемане автоматично се задава срок за връщане.

 Функционалности:
 Показване на всички книги
 Добавяне на нови книги (Librarian)
Заемане на книги (Student)
Връщане на книги (Student)
Проверка за просрочване
Обработка на грешки чрез изключения
Обработка на изключения
Заемане на вече заета книга
Връщане на книга, която не е заета
Опит за достъп до несъществуваща книга

 Използвани ООП принципи:
Инкапсулация – private/protected полета и свойства
Наследяване – Student и Librarian наследяват User
Полиморфизъм – override на методи
Абстракция – абстрактен клас User

Стартиране на проекта:
Отворете проекта във Visual Studio
Стартирайте конзолното приложение
Въведете име и тип потребител
Използвайте менюто за работа с библиотеката
